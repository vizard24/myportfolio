# **App Name**: Synapse Portfolio

## Core Features:

- Introduction Section: Display personal information including name, contact details, and a brief introduction.
- Project Showcase: Showcase projects with descriptions, tech stack details, and links to GitHub or live demos.
- Experience Timeline: Display a timeline of educational background and relevant work experience.
- Skills Section: Highlight technical skills with a visually appealing representation (e.g., skill bars).
- AI-Powered Resume Pitch Generator: A tool that utilizes a large language model to summarize your resume into different short pitches targeted to recruiters. Each summary targets different specialities. When a recruiter clicks a specialization button, they will see an ai generated blurb.

## Style Guidelines:

- Primary color: Dark navy blue (#24292F) for a professional look.
- Secondary color: Light gray (#E5E5E5) for backgrounds and subtle accents.
- Accent color: soft orange gradient (#FFA07A to #FFDAB9) for highlights, buttons, and interactive elements.
- Clean and modern sans-serif fonts for readability.
- Use a consistent set of icons to represent skills and technologies.
- Clean, single-page layout with smooth scrolling navigation.
- Subtle hover effects, parallax scrolling, and transitions to enhance interactivity and add visual depth.
